# 🛍️ Zyvora - Modern E-Commerce Frontend

Zyvora is a modern, responsive, and interactive e-commerce frontend built using HTML, CSS, Bootstrap, and JavaScript. It provides a clean shopping experience with a professional UI/UX design suitable for online retail platforms.

---

## ✨ Features

- 🔍 Smart Search Bar
- 📷 Camera Search Icon
- 🎤 Voice Search Icon
- 👤 Login & Registration Modal
- 🛒 Shopping Cart
- ❤️ Wishlist
- 📱 Fully Responsive Design
- 🎨 Modern UI/UX
- 🖼️ Product Carousel
- 🛍️ Product Categories
- 🔥 Trending Products
- 💰 Deals & Discounts
- ⭐ Featured Products
- 📦 Product Cards
- 📄 Product Details Section
- 📞 Contact Section
- 📍 Footer with Quick Links
- ⚡ Smooth Animations

---

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
